How URL Loading Works

Feature: Load a video configuration directly from a URL parameter

Format: index.html?id=NOTION_ID

Behavior:

  • Automatically loads the record on page load
  • No notification shown (silent load)
  • Fetches metadata from Notion + JSON from Supabase
