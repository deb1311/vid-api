üîç What Happens

When URL has ID parameter:

  1. Page loads
  2. JavaScript detects ?id= parameter
  3. Fetches JSON from Notion via Cloudflare Worker
  4. Loads data into editor (silently, no notification)
  5. Video preview and timeline render automatically

When URL has NO ID parameter:

  1. Page loads
  2. Shows empty editor
  3. User can click "Load from Notion" to browse records
