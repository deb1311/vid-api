Notion Database Reader - Test Page

How to get your Database ID:

  1. Open your Notion database in a web browser
  2. Copy the URL from the address bar
  3. The Database ID is the 32-character string in the URL
  4. Example: https://notion.so/workspace/abc123def456...?v=...
  5. The Database ID would be: abc123def456...
